# 🏠 House Price Prediction – End-to-End Machine Learning Project

## 📌 Project Overview

This project builds an end-to-end Machine Learning solution to predict house prices based on property features such as area, bedrooms, bathrooms, and location.

The goal is to:

* Accurately estimate house prices
* Identify key factors influencing property valuation
* Build a deployable prediction system
* Provide business insights from data

---

## 🎯 Business Problem

Real estate pricing is often inconsistent and manually driven.
This project aims to:

* Improve pricing accuracy
* Assist buyers and sellers in decision-making
* Automate property valuation
* Reduce human bias in pricing

---

## 📊 Dataset Description

The dataset includes:

* **Numeric Features:** Area, Bedrooms, Bathrooms, etc.
* **Categorical Features:** Location, Property Type
* **Target Variable:** Price

Data preprocessing steps:

* Handled missing values
* Removed duplicates
* Encoded categorical variables (One-Hot Encoding)
* Feature selection

---

##  Exploratory Data Analysis (EDA)

Performed:

* Correlation analysis
* Price distribution visualization
* Feature vs price relationship analysis
* Outlier detection

Key Findings:

* Area strongly impacts price
* Location significantly affects valuation
* Bedrooms moderately influence price

---

##  Model Development

### Models Used:

* Linear Regression
* Random Forest (optional improvement)

### Workflow:

1. Train-Test Split (80-20)
2. Feature encoding
3. Model training
4. Model evaluation

### Evaluation Metrics:

* MAE (Mean Absolute Error)
* RMSE (Root Mean Squared Error)
* R² Score
